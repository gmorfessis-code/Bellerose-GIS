# Interactive Parcel Map

An interactive web map displaying property parcels with detailed information including ownership, property classification, and municipal data.

## Features

- **Interactive Parcel Viewer**: Click on any parcel to view detailed information
- **Property Information**: Owner details, property classification, lot dimensions
- **External Links**:
  - Nassau County property records
  - Google Street View
- **Municipal Boundaries**: Shows village boundaries and adjacent municipalities
- **Responsive Design**: Works on desktop, tablet, and mobile devices

## Data Sources

- Property parcel data from Nassau County
- Municipal boundary layers
- NYS property classification codes

## Technology

- **QGIS**: Used for data processing and map creation
- **qgis2web**: Exports QGIS project to interactive web map
- **Leaflet**: JavaScript mapping library for web interactivity
- **OpenStreetMap**: Base map tiles

## Usage

Simply open the map in any web browser. Click on parcels to view property information.
